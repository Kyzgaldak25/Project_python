# Project_python
В данном блоке проведен глубокий анализ продаж с использованием библиотек Pandas и Matplotlib, а также интеграция внешних данных о погоде.

**Основные этапы работы:**

**Data Cleaning:** Преобразование форматов и работа с временными рядами.
  
**Группировка и визуализация:** Анализ динамики продаж и описание трендов на графиках.

**Поиск аномалий:** Выявление критических выбросов в данных о продажах.

**Углубленная фильтрация:** Поиск топовых товаров по сложным условиям (конкретные месяцы, дни недели и склад).

**External Data:** Парсинг и обработка архива погоды г. Астана с rp5.ru.

**Корреляционный анализ:** Сопоставление графиков продаж и средней температуры для поиска зависимостей.
